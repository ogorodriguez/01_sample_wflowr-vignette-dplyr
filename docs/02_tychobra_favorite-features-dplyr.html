across()

across() was created to supersede the functions that used to have as suffix *_at(), *_all() or *_if().

The data contains measurements for the basements, rooms, and living area of the houses in square feet. Let’s convert that to yards using a function.

ft_to_yrds <- function (x) {
  x / 9
}

The old way using mutate would have been:

ames_data %>%

The following examples shows how across() combines a mutate_if, and a mutate_at in one single, elegant line of code.

## Change to double the integer variables that ends with "_sf" only.

slice()

slice() and its helpers are now the way to select the top or lower values in a set and also to select random samples from the data.

slice()supersedes top_n(), sample_n(), sample_frac() functions due their confusing interpretation.
